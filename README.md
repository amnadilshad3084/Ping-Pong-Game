# 🏓 Ping Pong Game

A single-player Ping Pong game built in C++ using the Raylib 
graphics library. The player controls a paddle to bounce a ball — 
miss it and the game is over.

## 🛠️ Built With
- **Language:** C++
- **Library:** Raylib
- **IDE:** Visual Studio 2022

## 🎮 Features
- **Single-player gameplay** — control the paddle using keyboard keys
- **Real-time collision detection** between ball, paddle, and walls
- **Score tracking** — score increases with every successful hit
- **Game Over screen** displayed when the player misses the ball

## 🚀 How to Run
1. Make sure **Raylib** library is installed and linked in Visual Studio
2. Open `Ping Pong Project.sln` in Visual Studio 2022
3. Build the solution (`Ctrl + Shift + B`)
4. Run the game (`Ctrl + F5`)
5. Then use up down arrow keys to move the paddle up or down
6. Whenever ball is in contact with the paddle, score increments
7. If ball misses the paddle, game is over

## 📚 Concepts Used
- Programming Fundamentals concepts used
- Variables and data types for ball/paddle position and velocity
- Conditionals and loops for game logic and ball movement
- Functions for input handling, collision detection, and game updates
- GUI basics using Raylib for rendering graphics and managing the window
